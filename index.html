Enable Windows-Subsystem-Linux (WSL)

Here is the official documentation page.

Enable the 'Virtual Machine Platform' optional component

Open PowerShell as administrator and the command: dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

Then restart your PC.

Install a Linux Distro

At Microsoft Store App search for Linux and install any Linux distro, my suggestion is Ubuntu 18.04 LTS

Create your user

You will be asked to prove a new account username and password

(Optional) Install new Windows Terminal

Microsoft has a new Terminal with some cool features, you can get it here, but it's optional, you can use the Terminal you like most or even PowerShell or CmdPrompt. A very popular terminal emulator is Cmder
